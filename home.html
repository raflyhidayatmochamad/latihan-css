Cascading Style Sheets atau CSS adalah bahasa pemrograman berbasis aturan yang dikembangkan Oleh World Wide Web Consortium (W3C) pada tahun 1996. CSS dikembangkan karena HTML yang berfungsi u ntuk memformat halaman web tidak bisa menambahkan gaya atau tata letak tingkat lanjut. Misalnya, pada HTML versi 3.2, sudah diperkenalkan tag agar gaya teks bisa diubah. Namun, web designer amatir menyalahgunakannya dan tidak ada opsi fallback seperti yang sekarang tersedia di CSS. Jadi, pilihan terbaiknya adalah tetap memisahkan pemformatan (HTML) dan penataan gaya (CSS)
